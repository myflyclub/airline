## Project Overview
This is a Scala backend + JavaScript/HTML frontend project. The frontend uses jQuery AJAX, Chart.js, and Leaflet maps. The backend uses Play Framework with Scala. Be aware of Scala compilation rules (exhaustive pattern matching, implicit resolution, no dangling if-without-else).

## Code Changes
When fixing bugs or implementing changes, apply the simplest solution using existing methods/patterns first. Do not propose architectural changes or new abstractions unless explicitly asked.

## Database
For database/schema issues, provide one-time SQL migration scripts rather than DROP+CREATE or runtime approaches unless explicitly asked otherwise.

## Code Style
When editing HTML template files, preserve existing indentation style (tabs vs spaces). When editing Scala files, check for existing imports before adding new ones to avoid duplicates.
